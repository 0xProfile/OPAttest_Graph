# OPAttest_Graph

## Install

```
npm install -g @graphprotocol/graph-cli
```

## Init
```
graph init --studio credence
```

## Authenticate in cli
```
graph auth --studio <api-key-here>
```
## ENTER SUBGRAPH
```
cd credence
```

## Build subgraph
```
graph codegen && graph build
```

## Deploy
```
graph deploy --studio credence
```
